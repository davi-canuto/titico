Platform: Reservatório de Dopamina — a dark-themed content/course platform

## Navigation (top bar)
- Logo (top-left) + nav links: Início · Explorar · Comunidade · Menu
- Right side: search bar "Pesquisar conteúdos" + notification bell + user avatar + name
- Active nav item has red pill/outline style

## Home (Início)
- Hero carousel banner with CTA button (red, uppercase)
- "Continue assistindo" horizontal scroll row — video thumbnails with duration badge
- "Novidades" section — large card grid with colorful thumbnails
- "Trilha de X a Y" sections — course/learning path rows with module cards
- Module cards show: MÓDULO N label, title, thumbnail

## Explorar page
- Filter tabs at top: Continue assistindo · Novidades · Em Alta · Merecem Destaque · Top 10
- Video grid: thumbnail + play icon overlay + duration badge + title + category label
- "Adicionado há X dias/semana" freshness tag on each card
- Card types: Vídeo, Arquivo (downloadable), Live

## Comunidade page
- Social feed: posts by creator/admin
- "Conteúdo Exclusivo" badge on members-only posts
- Filter: Todos / Assinantes RD
- Post card: avatar + name + timestamp + content + image

## Visual patterns to replicate
- Dark bg (~#0d0d0d), red accent on active states and CTAs
- Section titles with red left accent or red underline
- Thumbnail cards: rounded corners, play overlay, duration bottom-right
- Tag badges: small pill shape, uppercase text
- Horizontal scroll rows with arrow navigation buttons

## How to apply
When designing dashboard, content pages, or admin screens for titico-app, reference these patterns:
- Netflix-style horizontal rows for content sections
- Card-based content grid for Explorar
- Section label + red accent + horizontal scroll = standard content section pattern
- Top navbar structure matches exactly what titico-app needs
